# Пирожковый Диспетчер (Sirius Plus)

Веб-приложение для отслеживания событий Sirius и автоматической записи при открытии регистрации.

## Приватность

- Email и пароль используются только для входа через `auth.sirius.online` в отдельном контексте браузера Playwright.
- Полученные токены Sirius и сохранённые данные входа хранятся в SQLite в зашифрованном виде с Fernet-ключом.
- Ключи, база данных, Web Push-ключ и runtime-данные исключены из Git через `.gitignore`.
- В проекте нет сторонней аналитики или отправки данных аккаунта на посторонние сервисы.

## Запуск

Для production используется Docker Compose. При первом старте контейнер создаёт ключ шифрования в подключённом каталоге данных.

Оригинальный экземпляр рассчитан на `sirius.rusanoff.ru` и требует приватный файл активации из persistent data volume. Публичный исходный код намеренно не является готовым к запуску production-экземпляром.

## Технологии

Python 3.12, FastAPI, Jinja2, SQLite, Playwright и Docker.
