# FCM-уведомления для Android

FCM нужен для системных уведомлений, когда Пирожковый Диспетчер закрыт или находится в фоне. Без него Android WebView не может надёжно получать события в фоне.

## 1. Создать Firebase-проект

1. Открой [Firebase Console](https://console.firebase.google.com/), создай проект.
2. Добавь Android-приложение с package name `ru.sonofstrange.siriusplus`.
3. Скачай `google-services.json` и положи его в `android/app/google-services.json`.
4. Собери новый APK. Gradle автоматически подключит Firebase, только когда этот файл существует.

Файл `google-services.json` исключён из Git и не должен попадать в архив релиза или репозиторий.

## 2. Дать серверу право отправлять FCM

1. В Firebase Console открой **Project settings → Service accounts**.
2. Создай новый private key в формате JSON.
3. На сервере сохрани файл как `/opt/sirius-data/firebase_service_account.json` с правами только для root:

```bash
install -m 600 /path/to/downloaded-firebase-key.json /opt/sirius-data/firebase_service_account.json
```

4. Пересобери и перезапусти контейнер. Docker уже монтирует `/opt/sirius-data` в `/app/data`, поэтому приложение увидит ключ автоматически.

```bash
cd /opt/sirius-plus
docker compose build --no-cache
docker compose up -d
```

При необходимости путь можно переопределить переменной `FCM_SERVICE_ACCOUNT_FILE`.

## 3. Проверить устройство

1. Установи APK 1.0.4 или новее, собранный с `google-services.json`.
2. Войди в аккаунт внутри приложения и разреши системные уведомления Android.
3. Сверни приложение.
4. На сайте в профиле нажми «Тест» в настройках уведомлений.

Сервер сохранит FCM-токен только после авторизованного входа в приложении. Один токен привязан к одному аккаунту и при входе в другой аккаунт переносится к нему.

## Тревога БПЛА

Сервер раз в 20 секунд получает состояние из API Sirius Radar/Плишкина. При переходе в состояние тревоги всем пользователям с сохранённым токеном отправляется FCM-сообщение на канал «Тревога» с системным alarm-звуком и вибрацией. При отбое отправляется обычное уведомление без специального сигнала.

Система не обходит пользовательские настройки Android: пользователь может отключить уведомления, звук, вибрацию или режим «Не беспокоить».
